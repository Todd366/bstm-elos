# Layer 1 — Reality Capture

**Rule:** Observation and interpretation must remain separate. Record "what actually happened" before asking "what does it mean."

This folder holds raw, unprocessed field evidence: photos, videos, interview notes, measurements, transaction logs, customer behavior notes, environmental conditions, direct quotes, timestamped events, technical logs, market conditions.

`offline_field_notebook.db` (generated by `scripts/init_offline_notebook.py`) is the low-connectivity fallback — a local SQLite cache that survives network drops in the field before syncing up to this folder / GitHub.

Nothing in this layer should contain interpretation, conclusions, or recommendations — that belongs in Layer 2 (`02_trial_intelligence/`).
