# BSTM ELOS Trial Migration Checklist

Before upgrading an old trial into ELOS standard:

## Required Structure

- [ ] YAML front matter exists
- [ ] Field Audit section exists
- [ ] Context section exists
- [ ] Guided Conversation Summary exists
- [ ] Key Quotes exists
- [ ] Assumption Registry exists
- [ ] Hypothesis exists
- [ ] Experiment exists
- [ ] Raw Evidence exists
- [ ] Interpretation exists
- [ ] Confidence Update exists
- [ ] Error Digestion exists
- [ ] Capability Check exists
- [ ] Ecosystem Mapping exists

## Validation

Run:

git add .
git commit

The ELOS validator must approve the trial.

## Rule

A trial is not considered upgraded until it passes the repository gate.
